# mjanowiak_vscode_agents

Simple Agents workspace for VS Code. Can also be used as a template for new projects.

## Documentation

- [Cloud Mode](docs/cloud-mode.md) – running this workspace in GitHub Codespaces or vscode.dev.

## Repository Structure

| Path | Purpose |
|------|---------|
| `src/` | Application source code |
| `tests/` | Unit and integration tests |
| `notebooks/` | Jupyter notebooks |
| `data/` | Datasets and data files |
| `assets/` | Static assets (images, etc.) |
| `docs/` | Project documentation |
| `dependencies/` | Vendored or pinned dependencies |
| `.github/instructions/` | GitHub Copilot coding instructions |

## Copilot Instructions

AI coding guidelines live in `.github/instructions/`:

- **`python.instructions.md`** – Python style, docstrings (Google format), typing, and testing conventions.
